Full Project Summary: Learn AI by Playing

Date: 15/05/2025

🎯 Project Goal

To build a bilingual (English/Spanish) educational game to teach basic AI concepts, using a web interface and AI model connection.

🧱 Project Structure

  • index.html – Main game interface.
  • script.js – Game logic.
  • style.css – Visual styles.
  • images/ – Baby face icons.
  • server/index.js – Node.js backend.
  • .env – Hugging Face token (not uploaded).
  • .gitignore – Hides sensitive files.

⚙️ Features

  • Language selector
  • Multiple-choice questions with baby faces
  • Connection to Hugging Face using gpt2
  • Tested using browser and Postman
  • Private GitHub repository
